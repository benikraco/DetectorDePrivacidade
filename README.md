# DetectorDePrivacidade

Este é um projeto de extensão do navegador que detecta várias informações relacionadas à privacidade ao visitar um site. Isso inclui detalhes sobre cookies (primeira e terceira partes, sessão e persistentes), dados de armazenamento local e URLs de terceiros.

## Características
- Contagem de cookies de primeira e terceira partes
- Contagem de cookies de sessão e persistentes
- Contagem de dados de armazenamento local
- Listagem de URLs de terceiros
- Detalhes dos cookies, incluindo nome e domínio
- A capacidade de limpar todos os dados de registro
